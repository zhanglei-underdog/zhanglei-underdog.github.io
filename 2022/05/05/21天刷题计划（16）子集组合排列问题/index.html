21天刷题计划（16）子集组合排列问题

发表于2022-05-05|更新于2022-05-05|21天刷题计划
|字数总计:2k|阅读时长:8分钟

子集

思路整理

  • 本题的元素唯一，且不能复选。属于常规的DFS框架
  • 需要注意的是，我们需要给DFS函数的参数列表加一个start参数，表示从数组的哪个元素开始遍历。用** track 记录根节点到每个节点的路径的值**。同时在前序位置把每个节点的路径值收集起来，完成回溯树的遍历就收集了所有子集。
  • 通过 start 参数控制树枝的遍历，避免产生重复的子集的主要原因是：使用 start 变量保证元素 nums[start] 之后只会出现 nums[start+1..] 中的元素
  • 我们在for循环外，如果提前发现不满足题目需要的路径，就不用再往下再继续遍历了，直接return，进行剪枝操作。

全排列Ⅱ

思路整理

  • 全排列一共有三种情况，无重无复，有重无复，无重可复
  • 有重有复在去重之后，属于无重可复
  • 本题属于较难的有重无复。我们有两种处理方案
    1. 用 prevNum 记录前一条树枝的值，就可以避免遍历值相同的树枝，从而避免产生相同的子树，最终避免出现重复的排列。用该方法，元素可以是无序的
    2. 保证相同元素在排列中的相对位置保持不变。输入 nums = [1,2,2’,2’’]，2’ 只有在 2 已经被使用的情况下才会被选择，2’’ 只有在 2’ 已经被使用的情况下才会被选择，这就保证了相同元素在排列中的相对位置保证固定。该方法需要使用辅助数组。
